# 내가 공부할것은 머지?
자신을 평가하고 어떤것을 공부할지 계획을 세우는 웹 시스템
<br/>
## 1. 목표와 기능
1)내가 어떠한것을 공부해야하는지 자신을평가한다.<br/>
2)계획을 세우고 캘런더에 추가한다.
<br/>
## 2. 기능
1)chat gpt 로 시험을쳐서 내가 생각하는게 맞는지 나의레벨을 측정한다.<br/>
2)계획대로 포스트잇을 올려서 자신이 얼마나했는지 보여준다.
<br/>
## 개발환경 및 배포
개발환경 - Html css js<br/>
배포 - http://sengwoong.github.io/ormi3Html
